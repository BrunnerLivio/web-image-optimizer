# web-image-optimizer

## Usage

```bash
./web-image-optimizer -i my-image.png -of jpg      
```

Will create a folder 
```
my-image
├── my-image_1200x.jpg
├── my-image_1500x.jpg
├── my-image_400x.jpg
├── my-image_770x.jpg
├── my-image_original.jpg
└── my-image_original.png
```

## Installation

1. Run `sudo apt-get install imagemagick jpegoptim`
2. Clone the repository and run the the command
